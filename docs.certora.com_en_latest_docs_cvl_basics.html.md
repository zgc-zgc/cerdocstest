# Basic Syntax[](https://docs.certora.com/en/latest/docs/cvl/basics.html#basic-syntax "Link to this heading")
## Comments[](https://docs.certora.com/en/latest/docs/cvl/basics.html#comments "Link to this heading")
Like Solidity files, whitespace in CVL files is not significant, and comments can be delimited by `/*` and `*/` or from `//` to the end of a line.
## Identifiers[](https://docs.certora.com/en/latest/docs/cvl/basics.html#identifiers "Link to this heading")
Identifiers in CVL have the same format as Solidity identifiers.
## Literals[](https://docs.certora.com/en/latest/docs/cvl/basics.html#literals "Link to this heading")
CVL supports integer literals in decimal or hexadecimal. CVL does not support Solidity units or rational literals.
CVL also supports string literals.
