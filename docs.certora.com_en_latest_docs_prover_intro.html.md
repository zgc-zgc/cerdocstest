# Introduction[](https://docs.certora.com/en/latest/docs/prover/intro.html#introduction "Link to this heading")
This Reference Manual contains detailed and comprehensive information about the Certora Prover. The Reference Manual is intended to describe what the Prover does, in contrast to the [Certora User’s Guide](https://docs.certora.com/en/latest/docs/user-guide/index.html) which explains how to use the Prover to accomplish particular goals.
